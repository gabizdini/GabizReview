# GabizReview

Portfólio pessoal de reviews de livros, compartilhável com amigos via link.

Quer conhecer? Acesse --> https://gabizreview.vercel.app/

## O que é

Um site para registrar reviews de livros com notas de 1 a 5 (incluindo meio estrela), capas e favoritos ("queridinhos").

## Funcionalidades

- **Explorar reviews** - Busca, filtros por nota e favoritos, ordenação, com persistência via URL
- **Ver detalhes** - Página individual da review com capa, estrelas, favorito e conteúdo
- **Painel admin** - CRUD completo (criar, editar, excluir reviews)
